# TinPDF

TinPDF is a React Native + Expo mobile app for converting images into PDF files.

## Current Milestone

- Expo SDK 54 project setup
- TinPDF app name and metadata
- Five-tab mobile shell: Home, Tools, Files, Scan, Settings
- Pick multiple images from the device library
- Scan/capture pages directly with the camera
- Continue capturing multiple pages into one PDF draft
- Append images to the current PDF draft
- Reorder selected images before export
- Preview selected pages before export
- Create an A4 PDF from selected images
- Save generated PDFs into the app document folder
- Open generated PDFs with the device PDF viewer
- Share or delete generated PDFs from the Files tab

## Run Locally

```powershell
npm.cmd install
npx.cmd expo start --localhost --port 8082
```
